# Banka – klijenti (Swing)

Jednostavna desktop aplikacija za evidenciju klijenata banke (Swing).
Funkcionalnosti: dodavanje/brisanje klijenata, prikaz broja klijenata, prosjeka, min/max stanja te sortiranje (A–Z / Z–A).

## Zahtjevi
- Java 21 (ili kompatibilno JDK 17+)
- Ant
- JUnit 4.13.2 i hamcrest-core 1.3 (u `lib/`)
